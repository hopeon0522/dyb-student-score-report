# DYB 학생 성장 리포트

영어 시험 성적표를 업로드하고 학생별 영역 점수, 총점, 전국·캠퍼스 석차 변화를 확인하는 상담용 웹사이트입니다.

## 주요 기능

- 학생별 Listening, Grammar, Reading 및 총점 변화 표시
- 전국 석차와 캠퍼스 석차 비교
- 시험에 응시하지 않은 기간은 `데이터 없음`으로 표시
- 파일명 `2026.02 중등 1차 형성평가.xls`을 `중등 1차 형성평가(26/02)`로 자동 변환
- DYB 성적표 형식의 HTML 기반 `.xls` 파일 업로드 지원
- 모바일 및 데스크톱 반응형 화면

## 로컬 실행

Node.js 22.13 이상과 pnpm이 필요합니다.

```bash
pnpm install
pnpm dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 빌드 확인

```bash
pnpm build
```

## 개인정보 주의

이 프로젝트는 학생 이름, 수험번호 및 성적을 다룰 수 있습니다. 저장소는 비공개로 유지하고 원본 성적표를 커밋하지 마세요. `참조` 폴더와 `.xls` 파일은 Git에서 제외됩니다.
